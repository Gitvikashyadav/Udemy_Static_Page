Udemy Static Page: This project is a static Udemy-style web page built with HTML, CSS, and JavaScript. It features a navigation bar, a banner carousel, course cards, a login/register popup, and a footer. The design aims to resemble the Udemy website.

Project Structure:

  Udemy_Static_Page:
  │
  ├── Udemy_Static_Page.html # This is the main HTML file that sets up the page structure.
  
  ├── Udemy.css # This CSS file handles the styling and makes the design responsive.
  
  ├── Udemy.js # This JavaScript file adds interactions and animations.
  
  └── image/ # This folder contains all the images used in the project.
Features

Navigation Bar – Dropdown menu and search box.
Auto Banner Carousel – Slides change every 5 seconds.
Login & Register Forms – Animated popup forms.
Responsive Layout – Works on desktop and mobile.
Footer Section – Styled like Udemy’s dark footer.
How to Run Step 1. Download or copy all files into one folder. Step 2. Open Udemy_Static_Page.html in any web browser (like Chrome or Edge). Step 3. That's it! You will see a completely static Udemy-style webpage.
